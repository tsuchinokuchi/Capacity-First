# CapacityFirst Mobile App - プロジェクト進捗状況

**最終更新**: 2026-01-16

## 現在のステータス: タスク再編集機能 実装完了

以下のコア機能の実装が完了しました。
1.  **Daily画面**: 今日のタスクのみを表示・管理。
2.  **Weekly画面**: 一週間のタスクを日付ごとに管理。
3.  **Pool画面**: 「いつかやる」未定タスクを管理。
4.  **データストア**: AsyncStorageによる完全な永続化。
5.  **再編集**: タスクのタイトル編集と日付変更（リスケジュール）。
6.  **キャパシティ管理**: 見積もり時間の設定と、日ごとの合計時間表示。
7.  **設定画面**: 1日のキャパシティ設定とデータ管理機能。

これにより、「自分の持ち時間（キャパシティ）を意識しながら、タスクを計画・実行する」というアプリのコアコンセプトが実現されました。

## 決定事項 (要件定義結果)

### 1. アプリの方向性
- **スタンドアローン**: PC版との連携は初期フェーズでは行わない。
- **オフラインファースト**: ローカルDBで完結し、爆速で動作する。

### 2. 主要機能・UI
- **Swipe Navigation**: 左右フリックで以下の画面を切り替え。
    - **[Pool]** (左/右 設定可) ⇔ **[Daily (Home)]** ⇔ **[Weekly]**
- **Weekly View**: 週間予定表を実装。
- **Dark Mode**: 標準対応。

### 3. 技術スタック
- **Framework**: React Native (Expo)
- **State**: Zustand
- **Storage**: react-native-mmkv
- **UI**: React Native Paper
