# CapacityFirst Mobile App - プロジェクト進捗状況

3.  **最終更新**: 2026-01-22

## 現在のステータス: 機能強化 & UX向上 (繰り返し・一括操作) 完了

以下のコア機能の実装が完了しました。
1.  **Daily画面**: 今日のタスクのみを表示・管理。Overdueタスク（期限切れ）の表示に対応。
2.  **Weekly画面**: 一週間のタスクを日付ごとに管理。フィルター（繰り返し/タグ）とキャパシティ表示に対応。
3.  **Pool画面**: 「いつかやる」未定タスクを管理。タグフィルターとタスク数表示に対応。
4.  **データストア**: AsyncStorageによる完全な永続化。
5.  **再編集**: タスクのタイトル編集と日付変更（リスケジュール）。
6.  **キャパシティ管理**: 見積もり時間の設定と、日ごとの合計時間表示。
7.  **設定画面**: 設定メニューの改装（タグ管理、並び替え設定、繰り返しルール管理）。
8.  **キャパシティ超過アラート**: 1日の上限を超えるタスク追加時に警告を表示。
9.  **UI/UX改善**: テーマカラー (Deep Blue & Teal) の適用、タスクソート機能、未完了タスクの視認性向上。
10. **繰り返しタスク**: 「毎日」「毎週」「カスタム」に対応。自動補充機能と「繰り返し停止（履歴保持）」機能の実装。
11. **[NEW] 一括編集機能**: Pool/Daily/Weekly全画面で、長押しによる複数選択と一括日付変更（Schedule）を実装。
12. **[NEW] Overdueタスク**: Daily画面上部に、過去の未完了タスクをまとめて表示・処理するアコーディオンを追加。

これにより、ただタスクを登録するだけでなく、「溜まったタスクを効率的に整理・消化する」フローが確立されました。

## 決定事項 (要件定義結果)

### 1. アプリの方向性
- **スタンドアローン**: PC版との連携は初期フェーズでは行わない。
- **オフラインファースト**: ローカルDBで完結し、爆速で動作する。

### 2. 主要機能・UI
- **Swipe Navigation**: 左右フリックで以下の画面を切り替え。
    - **[Pool]** (左/右 設定可) ⇔ **[Daily (Home)]** ⇔ **[Weekly]**
- **Weekly View**: 週間予定表を実装。
- **Dark Mode**: 標準対応。

### 3. 技術スタック
- **Framework**: React Native (Expo)
- **State**: Zustand

- **UI**: React Native Paper
