# Phase 2: QuickAdd設定手順（更新版）

## スクリプトの配置

スクリプトは既に `.obsidian/scripts/` フォルダに配置済みです：
- `タスク追加.js`
- `タスク移動.js`

## QuickAddの設定手順

### ステップ1: 「タスク追加」ChoiceでMacroを設定

1. QuickAddの設定画面で「タスク追加」Choiceを選択
2. 画面の上部にあるボタンから「⚡」（Macro）ボタンをクリック
   - または、Typeを「Macro」に変更
3. 「User Scripts」セクションで：
   - 「Start typing script name...」と書かれた入力欄に「タスク追加」と入力
   - または「Browse」ボタンをクリックして `タスク追加.js` を選択
4. 「Add」ボタンをクリック
5. 「Save」をクリック

### ステップ2: 「タスク移動」Choiceを作成（新規の場合）

1. 「Add Choice」ボタンをクリック
2. Nameに「タスク移動」と入力
3. 上部の「⚡」（Macro）ボタンをクリック
4. 「User Scripts」セクションで：
   - 「Start typing script name...」と書かれた入力欄に「タスク移動」と入力
   - または「Browse」ボタンをクリックして `タスク移動.js` を選択
5. 「Add」ボタンをクリック
6. 「Save」をクリック

## 使用方法

### コマンドパレットから実行
1. `Ctrl+P` (Windows) または `Cmd+P` (Mac) でコマンドパレットを開く
2. 「QuickAdd: タスク追加」または「QuickAdd: タスク移動」を検索
3. 選択して実行

## 動作確認

1. 「QuickAdd: タスク追加」を実行
2. タスク名、所要時間、ジャンル、日付を入力
3. 容量チェックが動作することを確認
4. スケジュールファイルにタスクが追加されることを確認

## トラブルシューティング

### スクリプトが見つからない場合
- `.obsidian/scripts/` フォルダにスクリプトが配置されているか確認
- Obsidianを再起動してみる

### エラーが発生する場合
- Obsidianのコンソール（`Ctrl+Shift+I`）でエラーメッセージを確認
- スクリプトのパスが正しいか確認
