# タスクジャンル設定

このファイルはCapacity-firstで使用するジャンル選択肢を定義しています。

## ジャンル一覧

以下のジャンルが使用可能です。編集する場合は、このファイルの配列部分を変更してください。

```javascript
const TASK_GENRES = [
  "デスクワーク",
  "売場作業",
  "顧客対応",
  "定型作業",
  "学習",
  "プライベート"
];
```

## 使用方法

各ダッシュボードファイルで以下のように使用します：

```javascript
// ジャンル設定ファイルを読み込む
const genreConfigPath = "03.ツェッテルカステン/030.データベース/タスク管理/ジャンル設定.md";
const genreConfigContent = await dv.io.load(genreConfigPath);
const genreMatch = genreConfigContent.match(/const TASK_GENRES = \[([\s\S]*?)\];/);
const genres = genreMatch 
  ? genreMatch[1]
      .split(',')
      .map(g => g.trim().replace(/^["']|["']$/g, ''))
      .filter(g => g)
  : ["デスクワーク", "売場作業", "顧客対応", "定型作業", "学習", "健康", "趣味", "その他プライベート"];
```
