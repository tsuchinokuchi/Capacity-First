# 🔁 毎月の繰り返しタスク

毎月実行するタスクを定義します。

```button
name 🔁 繰り返しタスクを展開
type command
action QuickAdd: 繰り返しタスク展開
```

## タスクフォーマット

```markdown
- [ ] タスク名 #ジャンル ⏱️ 60 🔁 every month on the 1st
```
