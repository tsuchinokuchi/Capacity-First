# 📋 タスクプール（未スケジュールタスク）

ここに、まだ日付が決まっていないタスクを記載します。

## タスクの追加方法

- QuickAddの「タスク追加」コマンドを使用
- または、直接このファイルに追加

## タスクフォーマット

```markdown
- [ ] タスク名 #ジャンル ⏱️ 60
```

### ジャンル一覧
- `#デスクワーク` - 資料作成、報告書など
- `#売場作業` - 品出し、売場整理など
- `#顧客対応` - 接客、問い合わせ対応など
- `#定型作業` - 週報、月報など
- `#学習` - 学習関連
- `#プライベート


### 所要時間
- `⏱️ 15` - 15分
- `⏱️ 30` - 30分
- `⏱️ 60` - 60分（1時間）
- など
